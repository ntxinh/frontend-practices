# tesla-landing-page

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.0.11. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

# How to run

```bash
bun run watch
bun run build

cd public
python3 -m http.server 3000
```
